# AI Researcher

## Что за задача?

Основная задача заключается в автоматизации научных процессов с применением агентных методов и LLM. Конкретно планируется реализация агентной системы для оптимизации процесса проведения исследовани засчет агентов по декомпозиции задачи и кодогенерации.

## Для кого?

Данный проект будет полезен для тех, кто активно проводит научные исследования и хочет автоматизировать этапы работы с кодом на своем пути.

## Какие боли?

Проект закрывает боли по ручному написанию и реализацию концепций, возникающих во время исследования.

## Что сделает PoC на демо?

На демо системы будет возможность увидеть процесс работы агентов декомпозиции задачи и кодогенерации.

## Что не сделает PoC?

- Проработанный пользовательский интерфейс
- Функционал для оформления решения
- Не гарантирует 100% точность решения с точки зрения оптимизации
